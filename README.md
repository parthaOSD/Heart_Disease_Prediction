## Heart Disease Prediction

## Overview
   This project illustrates the machine learning technique for predicting Heart Disease. 

## Dataset Used
   Kaggle Heart Disease Dataset

## Objective
   The objective of this project is to predict the likelihood of heart disease based on health realated factors using Machine Learning techniques

## Machine Learning Model Used
   Logistic Regression Model

## Evaluation Metrics
   1)Accuracy Score

## Results
   By using Logistic Regression, we get the accuracy score. We also predict if a patient have heart disease or not.

## Technologies Used
   1) Python
   2) Numpy
   3) Pandas
   4) Scikit_learn
